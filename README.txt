CLASSMATE — SECTION 6 ATTENDANCE TRACKER

This version is general for any Section 6 student:
- First-time setup asks for the student's name.
- The actual Section 6 timetable from the supplied Excel file is built in.
- Each person's attendance stays in their own browser/device.
- History lets users correct past attendance.
- Mobile responsive and installable like an app when hosted over HTTPS.

DEPLOY FREE WITH GITHUB PAGES
1. Create a GitHub repository such as classmate-attendance.
2. Upload ALL files from this folder to the repository root.
3. Open repository Settings > Pages.
4. Under Build and deployment choose Deploy from a branch.
5. Select main and /(root), then Save.
6. Share the HTTPS link GitHub gives you.

PHONE
Android/Chrome: open the hosted link > menu > Add to Home screen / Install app.
iPhone/Safari: open the hosted link > Share > Add to Home Screen.

IMPORTANT
The workbook supplied contains the Section 6 timetable. To support Sections 1–5 correctly,
their timetable data/files are also required.

Attendance is local to each browser/device and does not sync between phone and laptop.
Cloud sync would require authentication and a cloud database.
